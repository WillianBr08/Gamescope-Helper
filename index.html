Gamescope Helper

Uma interface gráfica rápida e nativa em Python e PyQt6 para gerar opções de inicialização do Gamescope para a Steam.

Baixar AppImage (Linux) Compatível com CachyOS, Arch, Ubuntu e Fedora

Como executar: 

chmod +x Gamescope_Helper-x86_64.AppImage
./Gamescope_Helper-x86_64.AppImage
